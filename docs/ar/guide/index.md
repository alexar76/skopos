# توثيق SKOPOS

مرحبًا بك في دليل مشغّل SKOPOS. استخدم علامات التبويب أعلاه للانتقال بين الإعداد والنشر والتكوين والاستخدام اليومي.

## ماذا يفعل SKOPOS

يجمع SKOPOS **سجلات وصول nginx** من خوادمك عبر SSH، ويخزّنها في قاعدة SQLite أو PostgreSQL محلية، ويعرض التحليلات ومركز الأمان مع تقارير مدعومة بالذكاء الاصطناعي.

## روابط سريعة

| المهمة | المكان |
|------|--------|
| الإعداد لأول مرة | **البدء السريع** في الشريط الجانبي أو الشريط العلوي |
| SSH والسجلات | **الإعدادات** → `servers.yaml` |
| لوحات حركة المرور | **التحليلات** (الصفحة الرئيسية) |
| فحوصات الأمان | **الأمان** |
| وكيل AI | زر الدردشة العائم (أسفل اليمين) |

## Pages at a glance

| المهمة | المكان |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
