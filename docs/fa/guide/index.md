# مستندات SKOPOS

به راهنمای اپراتور SKOPOS خوش آمدید. از تب‌های بالا برای جابجایی بین راه‌اندازی، استقرار، پیکربندی و استفاده روزانه استفاده کنید.

## SKOPOS چه می‌کند

SKOPOS **لاگ‌های دسترسی nginx** را از سرورهای شما از طریق SSH جمع‌آوری می‌کند، در SQLite یا PostgreSQL محلی ذخیره می‌کند و تحلیل‌ها به‌همراه مرکز امنیت با گزارش‌های کمکی AI نمایش می‌دهد.

## پیوندهای سریع

| کار | محل |
|------|--------|
| راه‌اندازی اولیه | **شروع سریع** در نوار کناری یا بالا |
| SSH و لاگ‌ها | **تنظیمات** → `servers.yaml` |
| داشبوردهای ترافیک | **تحلیل‌ها** (صفحه اصلی) |
| اسکن امنیتی | **امنیت** |
| عامل AI | دکمه چت شناور (پایین راست) |

## Pages at a glance

| کار | محل |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
