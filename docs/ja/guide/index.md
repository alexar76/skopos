# SKOPOS ドキュメント

SKOPOS オペレーターガイドへようこそ。上部のタブでセットアップ、デプロイ、設定、日常運用を切り替えられます。

## SKOPOS の機能

SKOPOS は SSH 経由でサーバーから **nginx アクセスログ** を収集し、ローカルの SQLite または PostgreSQL に保存し、AI 支援レポート付きの分析とセキュリティセンターを表示します。

## クイックリンク
