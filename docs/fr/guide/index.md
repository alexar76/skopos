# Documentation SKOPOS

Bienvenue dans le guide opérateur SKOPOS. Utilisez les onglets ci-dessus pour passer entre installation, déploiement, configuration et usage quotidien.

## Ce que fait SKOPOS

SKOPOS collecte les **journaux d'accès nginx** de vos serveurs via SSH, les stocke dans SQLite ou PostgreSQL local, et affiche des analyses plus un Centre de sécurité avec rapports assistés par IA.

## Liens rapides

| Tâche | Où |
|------|--------|
| Première installation | **Démarrage rapide** dans la barre latérale ou supérieure |
| SSH et journaux | **Paramètres** → `servers.yaml` |
| Tableaux de trafic | **Analytique** (accueil) |
| Analyses de sécurité | **Sécurité** |
| Agent IA | Bouton de chat flottant (en bas à droite) |
