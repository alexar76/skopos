# Dokumentasi SKOPOS

Selamat datang di panduan operator SKOPOS. Gunakan tab di atas untuk beralih antara penyiapan, deployment, konfigurasi, dan penggunaan harian.

## Apa yang dilakukan SKOPOS

SKOPOS mengumpulkan **log akses nginx** dari server Anda via SSH, menyimpannya di SQLite atau PostgreSQL lokal, dan menampilkan analitik serta Pusat Keamanan dengan laporan berbantuan AI.

## Tautan cepat

| Tugas | Lokasi |
|------|--------|
| Penyiapan pertama | **Mulai Cepat** di sidebar atau bar atas |
| SSH & log | **Pengaturan** → `servers.yaml` |
| Dasbor lalu lintas | **Analitik** (beranda) |
| Pemindaian keamanan | **Keamanan** |
| Agen AI | Tombol chat mengambang (kanan bawah) |

## Pages at a glance

| Tugas | Lokasi |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
