# Tài liệu SKOPOS

Chào mừng đến hướng dẫn vận hành SKOPOS. Dùng các tab phía trên để chuyển giữa thiết lập, triển khai, cấu hình và sử dụng hàng ngày.

## SKOPOS làm gì

SKOPOS thu thập **nhật ký truy cập nginx** từ máy chủ qua SSH, lưu vào SQLite hoặc PostgreSQL cục bộ, và hiển thị phân tích cùng Trung tâm bảo mật với báo cáo hỗ trợ AI.

## Liên kết nhanh

| Tác vụ | Vị trí |
|------|--------|
| Thiết lập lần đầu | **Bắt đầu nhanh** trên thanh bên hoặc thanh trên |
| SSH & nhật ký | **Cài đặt** → `servers.yaml` |
| Bảng điều khiển lưu lượng | **Phân tích** (trang chủ) |
| Quét bảo mật | **Bảo mật** |
| Tác tử AI | Nút chat nổi (góc dưới phải) |

## Pages at a glance

| Tác vụ | Vị trí |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
