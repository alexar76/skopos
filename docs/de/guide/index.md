# SKOPOS-Dokumentation

Willkommen im SKOPOS-Betreiberhandbuch. Nutzen Sie die Tabs oben für Setup, Deployment, Konfiguration und den täglichen Betrieb.

## Was SKOPOS leistet

SKOPOS sammelt **nginx-Zugriffslogs** per SSH von Ihren Servern, speichert sie in SQLite oder PostgreSQL lokal und zeigt Analysen sowie ein Security Center mit KI-gestützten Berichten.
