# Documentazione SKOPOS

Benvenuto nella guida operatore SKOPOS. Usa le schede in alto per passare tra setup, deployment, configurazione e uso quotidiano.

## Cosa fa SKOPOS

SKOPOS raccoglie **log di accesso nginx** dai server via SSH, li salva in SQLite o PostgreSQL locale e mostra analitiche più un Security Center con report assistiti da IA.

## Link rapidi

| Attività | Dove |
|------|--------|
| Prima configurazione | **Avvio rapido** nella barra laterale o superiore |
| SSH e log | **Impostazioni** → `servers.yaml` |
| Dashboard traffico | **Analitiche** (home) |
| Scansioni sicurezza | **Sicurezza** |
| Agente IA | Pulsante chat flottante (in basso a destra) |
