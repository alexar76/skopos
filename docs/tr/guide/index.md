# SKOPOS Dokümantasyonu

SKOPOS operatör kılavuzuna hoş geldiniz. Kurulum, dağıtım, yapılandırma ve günlük kullanım arasında geçmek için üstteki sekmeleri kullanın.

## SKOPOS ne yapar

SKOPOS, sunucularınızdan SSH ile **nginx erişim günlüklerini** toplar, yerel SQLite veya PostgreSQL'e kaydeder ve AI destekli raporlarla analitik ve Güvenlik Merkezi sunar.

## Hızlı bağlantılar

| Görev | Konum |
|------|--------|
| İlk kurulum | Kenar çubuğu veya üst çubukta **Hızlı Başlangıç** |
| SSH ve günlükler | **Ayarlar** → `servers.yaml` |
| Trafik panoları | **Analitik** (ana sayfa) |
| Güvenlik taramaları | **Güvenlik** |
| AI aracısı | Sağ altta yüzen sohbet düğmesi |

## Pages at a glance

| Görev | Konum |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
