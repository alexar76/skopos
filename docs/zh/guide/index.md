# SKOPOS 文档

欢迎使用 SKOPOS 操作员指南。使用上方标签页可在设置、部署、配置和日常使用之间切换。

## SKOPOS 的功能

SKOPOS 通过 SSH 从服务器收集 **nginx 访问日志**，存储到本地 SQLite 或 PostgreSQL 数据库，并提供分析面板及带 AI 辅助报告的安全中心。

## 快速链接

| 任务 | 位置 |
|------|--------|
| 首次设置 | 侧边栏或顶部栏中的 **快速入门** |
| SSH 与日志 | **设置** → `servers.yaml` |
| 流量仪表板 | **分析**（首页） |
| 安全扫描 | **安全** |
| AI 代理 | 右下角浮动聊天按钮 |
