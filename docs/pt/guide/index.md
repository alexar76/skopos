# Documentação SKOPOS

Bem-vindo ao guia do operador SKOPOS. Use as abas acima para alternar entre configuração, implantação, ajustes e uso diário.

## O que o SKOPOS faz

O SKOPOS coleta **logs de acesso nginx** dos seus servidores via SSH, armazena-os em SQLite ou PostgreSQL local e exibe análises e um Centro de Segurança com relatórios assistidos por IA.
