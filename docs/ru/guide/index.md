# Документация SKOPOS

Добро пожаловать в руководство оператора. Вкладки выше — обзор, развёртывание, настройка, использование и раздел про HTTP-логи.

## Что делает SKOPOS

SKOPOS собирает **access-логи nginx** с серверов по SSH, сохраняет их в SQLite/PostgreSQL и показывает аналитику плюс Security Center с AI-отчётами.

## Быстрые ссылки

| Задача | Где |
|------|--------|
| Первый запуск | **Быстрый старт** в сайдбаре или верхнем меню |
| SSH и логи | **Настройки** → `servers.yaml` |
| Трафик | **Аналитика** (главная) |
| Сканирование | **Безопасность** |
| AI-агент | Кнопка чата справа внизу |

## Страницы интерфейса

| Задача | Где |
|------|--------|
| **Быстрый старт** | Мастер из 6 шагов — сервер, SSH, пароль, сбор, скан |
| **Аналитика** | Дашборды трафика, AI-брифинг, 7 вкладок (Обзор → Система) |
| **Безопасность** | 9 вкладок — скор, AI-отчёт, порты, стуки, 3D-карта, аудит |
| **История сканов** | Таймлайн, тренды, сравнение двух сканов, журнал |
| **Настройки** | Пароль, БД, авто-скан, Telegram, SSH-ключи, fleet YAML |
| **Документация** | Это руководство — 20 языков, скриншоты интерфейса |

> Откройте **Документация** в сайдбаре. Вкладка **Использование** описывает каждую страницу и подраздел со скриншотами.
