# SKOPOS दस्तावेज़

SKOPOS ऑपरेटर गाइड में आपका स्वागत है। सेटअप, डिप्लॉयमेंट, कॉन्फ़िगरेशन और दैनिक उपयोग के बीच जाने के लिए ऊपर के टैब का उपयोग करें।

## SKOPOS क्या करता है

SKOPOS SSH के माध्यम से आपके सर्वरों से **nginx एक्सेस लॉग** एकत्र करता है, उन्हें स्थानीय SQLite या PostgreSQL डेटाबेस में संग्रहीत करता है, और AI-सहायता प्राप्त रिपोर्ट के साथ एनालिटिक्स तथा Security Center प्रदर्शित करता है।

## त्वरित लिंक

| कार्य | स्थान |
|------|--------|
| पहली बार सेटअप | साइडबार या टॉप बार में **त्वरित प्रारंभ** |
| SSH और लॉग | **सेटिंग्स** → `servers.yaml` |
| ट्रैफ़िक डैशबोर्ड | **एनालिटिक्स** (होम) |
| सुरक्षा स्कैन | **सुरक्षा** |
| AI एजेंट | फ़्लोटिंग चैट बटन (नीचे दाएँ) |

## Pages at a glance

| कार्य | स्थान |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
