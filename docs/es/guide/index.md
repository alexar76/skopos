# Documentación SKOPOS

Bienvenido a la guía del operador. Use las pestañas para ir a despliegue, configuración, uso y logs HTTP.

## Qué hace SKOPOS

SKOPOS recopila **logs de acceso nginx** por SSH, los guarda en SQLite/PostgreSQL y muestra analítica más un Security Center con informes AI.

## Enlaces rápidos

| Tarea | Dónde |
|------|--------|
| Primer arranque | **Inicio rápido** en la barra lateral o superior |
| SSH y logs | **Ajustes** → `servers.yaml` |
| Tráfico | **Analítica** (inicio) |
| Seguridad | **Seguridad** |
| Agente AI | Botón de chat (abajo a la derecha) |

## Pages at a glance

| Tarea | Dónde |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
