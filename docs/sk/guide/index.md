# Dokumentácia SKOPOS

Vitajte v príručke operátora SKOPOS. Pomocou kariet hore prepínajte medzi nastavením, nasadením, konfiguráciou a každodenným používaním.

## Čo SKOPOS robí

SKOPOS zbiera **nginx access logy** zo serverov cez SSH, ukladá ich do lokálneho SQLite alebo PostgreSQL a zobrazuje analytiku a Security Center s AI reportmi.

## Rýchle odkazy

| Úloha | Kde |
|------|--------|
| Prvé nastavenie | **Rýchly štart** v bočnom alebo hornom paneli |
| SSH a logy | **Nastavenia** → `servers.yaml` |
| Traffic dashboardy | **Analytika** (domov) |
| Bezpečnostné skeny | **Bezpečnosť** |
| AI agent | Plávajúce tlačidlo chatu (vpravo dole) |

## Pages at a glance

| Úloha | Kde |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
