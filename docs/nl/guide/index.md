# SKOPOS-documentatie

Welkom bij de SKOPOS-operatorgids. Gebruik de tabbladen bovenin voor setup, deployment, configuratie en dagelijks gebruik.

## Wat SKOPOS doet

SKOPOS verzamelt **nginx-accesslogs** van uw servers via SSH, slaat ze op in lokale SQLite of PostgreSQL en toont analyses plus een Security Center met AI-ondersteunde rapporten.
