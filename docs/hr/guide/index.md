# SKOPOS dokumentacija

Dobrodošli u vodič za operatera SKOPOS-a. Koristite kartice gore za postavljanje, implementaciju, konfiguraciju i svakodnevnu upotrebu.

## Što SKOPOS radi

SKOPOS prikuplja **nginx access logove** s vaših poslužitelja putem SSH-a, pohranjuje ih u lokalni SQLite ili PostgreSQL i prikazuje analitiku te Security Center s AI izvješćima.

## Brze poveznice

| Zadatak | Gdje |
|------|--------|
| Prvo postavljanje | **Brzi početak** u bočnoj ili gornjoj traci |
| SSH i logovi | **Postavke** → `servers.yaml` |
| Nadzorne ploče prometa | **Analitika** (početna) |
| Sigurnosna skeniranja | **Sigurnost** |
| AI agent | Plutajući gumb za chat (dolje desno) |

## Pages at a glance

| Zadatak | Gdje |
|------|--------|
| **Quick Start** | Six-step wizard — server, SSH, password, collect, scan |
| **Analytics** | Traffic dashboards, AI briefing, 7 tabs (Overview → System) |
| **Security** | 9 tabs — score, AI report, ports, knocks, 3D map, audit |
| **Scan History** | Timeline, trends, compare two scans, log table |
| **Settings** | Password, DB, auto-scan, Telegram, SSH keys, fleet YAML |
| **Documentation** | This guide — 20 languages, embedded screenshots |

> Open **Documentation** in the sidebar for the full operator guide with screenshots. The **Usage** tab documents every page and tab in detail.
